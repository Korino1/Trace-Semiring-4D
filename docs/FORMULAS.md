# Формулы TS4 (краткий справочник)

1. Каноническая форма трассы:
   \langle (a_0,b_0,c_0) | ... | (a_k,b_k,c_k) \rangle

2. Композиция трасс (склейка границы):
   U∘V = \langle B_0|...|B_{k-1}|(B_k+C_0)|C_1|...|C_m \rangle

3. Сложение TS4:
   (A+B)(t)=A(t)+B(t)

4. Композиция TS4:
   (A∘B)(t)=\sum_{uv=t} A(u)B(v)

5. κ‑нормализация:
   Φ_κ: разбиение переполненных блоков так, что |B_i|_1 ≤ κ

6. Физические операции:
   odot_κ = Φ_κ ∘ ∘
   parallel_κ — послойная сумма + split_κ
   otimes_κ — подстановка τ↦V + Φ_κ
